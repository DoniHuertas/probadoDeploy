API's y Axios

¿Qué es una API?

API es la abreviatura de "Application Programming Interface", que es español se traduciría como "Interfaz de Programaciónd de Aplicaciones".

Se trata de un conjunto de reglas y protocolos que permiten que diferentes aplicaciones o componentes se comuniquen entre si.

¿Qué es AXIOS?

Axios es una biblioteca que se utiliza para realizar solicitudes HTTP. Puede ser usado tanto en el FE como en el BE.
